# CIFAR-10 Image Classification

## Overview

This project builds a deep learning model to classify images from the CIFAR-10 dataset. The model predicts the object present in an image such as airplane, car, dog, or ship.

## Dataset

The CIFAR-10 dataset contains 60,000 color images of size 32x32 in 10 classes:
Airplane, Automobile, Bird, Cat, Deer, Dog, Frog, Horse, Ship, Truck.

## Technologies Used

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib

## Model

A Convolutional Neural Network (CNN) is used to learn image features and classify them into the correct category.

## How to Run

Install libraries:
pip install -r requirements.txt

Train the model:
python train.py

## Result

The CNN model classifies CIFAR-10 images with good accuracy.
